Install paxmeet_icons

Flutter app

1

Add to pubspec.yaml under dependencies:

paxmeet_icons: git: url: https://github.com/letssuhail/paxmeet_icons.git ref: main
2

Install

flutter pub get
3

Use anywhere

Icon(PaxmeetIcons.home)

Website (Next.js / React)

1

Install from GitHub (needs repo read-access)

npm install github:letssuhail/paxmeet_icons
2

Allow the package to transpile - in next.config.mjs

transpilePackages: ['paxmeet_icons'],
3

Import the CSS once in app/layout.js

import "paxmeet_icons/css";
4

Use anywhere

import { PaxmeetIcon } from "paxmeet_icons";
<PaxmeetIcon name="home" />
