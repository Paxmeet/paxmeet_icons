# Changelog

## 0.0.1

- 542 icons shipped as a single `PaxmeetIcons` font.
- Flutter package (`PaxmeetIcons.*`) and web assets (`PaxmeetIcon` component + CSS classes) generated from the same font.
- Stroke-based icons normalized to filled outlines via picosvg.
- `const IconData` constants, tree-shaken in release builds.
- Self-contained reference page (`index.html`) with search and copy-to-clipboard.
